How to host your website! 12/12/2017

Github Offers free hosting!

To start hosting your website follow these steps:

1. Log on to your Github.io account

2. Create a new Repository with your username.github.io. Include a ReadMe document when starting.

3. Upload an index.html file with simple text for tesing purposes.

4. Go to the Settings within your github page. Check GitHub Pages section to see info about hosting

5. Be sure Your site is being published and have no warning messages.

Check your github website at your username.github.io

Now you have your website up!

6. Now setup your custom domain. Within Settings: Type in your custom domain. example: google.com

7. Logon to your Namecheap account.

8. Select Domain List > Advance DNS section.

9. Add these records : A record @ 192.30.252.153, A record @ 192.30.252.154, CNAME Record www username.github.io.

10. It should take about 30minutes for the changes to take effect. Enjoy your website!
